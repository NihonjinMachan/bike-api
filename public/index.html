Fetch station snapshot

  • A single endpoint is being used for fetching spanshots for both all stations at a given time as well as a specific station by kioskId for a given time
  • These functionalities are differentiated by the presence of specific query string params
  • To fetch snapshot for a specfic kiosk use: /api/v1/stations?kioskId={kioskID}&at={time} (sample kioskID = 3004, 3205)
  • To fetch snapshot for all kiosks use: /api/v1/stations?at={time}
  • NOTE: Please make sure to set the ('api-token': bikeroni123) key on the request headers
  • A postman collection is included in the source code for convenience
